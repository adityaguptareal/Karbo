# Karbo Market

**Karbo Market** is a web app that connects farmers with businesses looking to buy carbon credits. Farmers can list their land, and businesses can directly purchase carbon credits. Farmers earn based on the credits generated from their land, creating a transparent way to monetize sustainable practices.

---

## Features

- Farmers can register their land and carbon credits  
- Businesses can browse and buy carbon credits directly from farmers  
- Payments handled through **Razorpay**  
- Earnings for farmers are **distributed monthly** during the lock-in period  
- **Role-based signup** (Farmer / Business)  
- **Admin approval** for new users before they can list land or buy credits  
- JWT authentication for secure login  

---

## Tech Stack

- **Frontend:** React  
- **Backend:** Node.js with Express  
- **Database:** MongoDB  
- **Payments:** Razorpay  
- **Deployment:**  
  - Frontend: Firebase Hosting  
  - Backend: Render
